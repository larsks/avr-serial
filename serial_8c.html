Macro Definition Documentation

#define SERIAL_BPS   4800

Serial bitrate.

Defaults to 4800, but the code will support up to 9600bps with the clock at 1Mhz, and up to 57.6Kbps with the clock at 8Mhz.

#define TICKS_PER_BIT   (F_CPU/SERIAL_BPS/PRESCALER_VAL)

The number of timer tickets that correspond to a single bit.

We load this value into the OCR0A register.

Function Documentation

ISR ( TIM0_COMPA_vect  )

Timer0 output compare match interrupt routine.

This is responsible for sending bits out SERIAL_TXPIN. If you have enabled SERIAL_PROVIDE_MILLIS, it is also responsible for incrementing the millis counter.

millis_t millis ( )

Return current millis counter.

The millis counter starts incrementing when you call serial_begin, and stops counting when you call serial_end.

void serial_init ( )

Initialize software serial support.

This sets up the appropriate DDR register, TCCR0A, TCCR0B, and sets OCR0A to a value appropriate for the selected clock speed and bitrate.
